Scope of Work Summary

Deliverables cover a custom Google Calendar integration and a custom‑designed WordPress site built specifically for Tri‑UU’s needs.

Custom Google Calendar Integration

1) Custom Calendar Display

  • I created a standalone calendar feature that plugs directly into your website.
  • On computers: a full month‑at‑a‑glance calendar grid with days, events, and month navigation.
  • On phones/tablets: a simple, mobile‑friendly 7‑day rolling list with week‑by‑week navigation.

2) Live Connection to Google Calendar

  • The calendar automatically pulls events from Google Calendar in real time.
  • You only add or edit events once in Google Calendar; the website reflects updates instantly.

3) Event Details

  • Event name and time (or “All Day”).
  • Location when provided.
  • Notes/description text with clickable links where appropriate.

4) User Navigation

  • Visitors can move between weeks or months with built‑in “Next/Previous” links.
  • View auto‑jumps to the calendar section so visitors don’t lose their place.

5) Automatic Mobile/Desktop Switching

  • The system detects device size and shows list or grid automatically—no extra setup required.

6) Error Handling & Safety

  • If Google Calendar can’t be reached, a clear message appears rather than a broken layout.
  • Event descriptions are processed safely so outside links open securely.

7) Extra Compatibility Work

  • A helper enhancement improves Simple Calendar plugin event displays so start times and descriptions appear consistently with your custom calendar.

Value Statement

This custom calendar system saves time by eliminating duplicate updates—Google Calendar becomes your single source of truth. Visitors get a clean, responsive experience on any device. Overall, it provides a professional and reliable way to showcase events, reducing manual work while giving your audience the information they need at a glance.

WordPress Website Setup — Scope of Work

Core Website Platform

  • Latest WordPress (6.8.2) on a secure Apache / PHP 8.3 server with MySQL 8 for stability, compatibility, and performance.

Theme & Design

  • Built on Hello Elementor and enhanced with Elementor Pro for editing flexibility.
  • Entire layout, structure, and visual presentation custom‑designed for Tri‑UU’s brand and goals—no pre‑made template.

Essential Plugins Installed & Configured

  • Yoast SEO – search engine optimization.
  • Google Analytics (MonsterInsights) – visitor analytics and reports.
  • WPForms Lite – contact forms.
  • OptinMonster – pop‑ups and lead capture.
  • Jetpack – security monitoring, backups, stats, additional blocks/widgets.
  • Image Optimizer – automatic compression and resizing for fast loads.
  • Simple Calendar – connects to Google Calendar and displays events.

Server & Hosting Setup

  • Performance tuned: 512 MB PHP memory, large file uploads, modern protocols (OpenSSL, cURL).
  • Remote connections enabled (forms, API calls, updates).
  • Secure defaults (no public error display, SOAP support, modern PHP).
